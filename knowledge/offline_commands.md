# Base local de comandos offline

Q: Como iniciar o copiloto técnico local?
A: Use `cerebro central`.

---

Q: Como reindexar documentos e código?
A: Use `cerebro index`.

---

Q: Como ativar reindexação incremental?
A: Use `cerebro watch`.

---

Q: Como iniciar a API local?
A: Use `cerebro api`.

---

Q: Como verificar se Ollama está disponível?
A: Rode `curl http://localhost:11434/api/tags`.

---

Q: Como iniciar o Ollama local?
A: Use `ollama serve`.

---

Q: Qual modelo recomendado para Mac 2017 i5 8GB?
A: Use `deepseek-coder:6.7b-instruct-q4_K_M` com contexto 1024.
